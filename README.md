## 프로젝트 소개
여행정보 요약 사이트로 공공데이터포털의 여행정보 api를 사용하여 여행정보를 수집하고 관련 기사를 chatGPT로 요약하여 제공합니다.

## 기술 스택
- Java
- Spring Boot
- Vue.js
